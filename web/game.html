This is our version of the Takuzu game that illustrates a 2-Dimensional grid with options below. The objective of the game is to fill the entire grid with asteroids and stars while respecting specific rules.

Game Rules

Fill the grid with either asteroids or stars, in such a way that:
- no three consecutive squares, horizontally or vertically, are the same image
- each row and column contains the same number of stars and asteroids.
Left-click in an empty square to set an asteroid :
Click again to set a star :
Then click again to set an empty square.
